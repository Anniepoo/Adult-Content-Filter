Aplikasi Facebook Android yang Ringan
Aplikasi facebook adalah aplikasi yang paling memberatkan hp android. Tidak mengherankan karena aplikasi facebook memiliki banyak sekali fitur, terutama fitur yang berjalan di background yang tanpa kita sadarai terus berjalan dan memakan memory. Contohnya fitur cek in yang yang memerlukan akses ke GPS, atau fitur push notifikasi yang secara tidak kita sadari terus terusan mengecek pembaruan dari akun facebook kita apakah ada pesan baru, atau pemberitahuan baru dan mengirimkannya ke aplikasi. Hal ini juga dapat menyebabkan baterai cepet habis. Dengan kata lain aplikasi facebook adalah salah satu aplikasi android yang paling boros RAM dan mengakibatkan boros baterai.
Untungnya baru baru ini facebook mengeluarkan aplikasi facebook terbaru yang memungkinkan kita tidak hanya menghemat baterai dan membuat kita hp android kita ringan, juga dapat menghemat penggunaan kuota internet saat menjelajahi facebook. Aplikasi gratis ini bernama : "Facebook Lite"
Kelebihan Facebook Lite Android
Keunggulan dari facebook lite seperti yang sudah dikatakan diatas adalah ringan, cepat dibuka, tidak boros baterai, tidak boros internet dan anti force close. Hal tersebut dikarenakan tidak terdapat banyak fitur yang disematkan, malah beberaoa pengguna menyebutnya mirip dengan aplikasi facebook untuk hp Java. Seperti fitur check in, tag, dan lainnya tidak terdapat disini, namun kita masih akan bisa mendapatkan fitur notifikasi / pemberitahuan jika ada pembaruan seperti like, update status, koment, mengirim pesan dan lainya.
Sementara untuk menghemat penggunaan data / kuata internet, aplikasi facebook lite telah mengkompress gambar menjadi sangat kecil, konsekuensinya kadang gambar menjadi blur dan tidak jelas.
Facebook memang sengaja mengeluarkan aplikasi untuk pengguna android kelas bawah yang hp androidnya sering hang kalau menggunakan aplikasi facebook. Jadi kalau anda menggunakan hp android terbaru dengan spesifikasi bagus yang disarankan menggunakan aplikasi facebook biasa.

Download RAM Manager Pro Apk - Aplikasi untuk memperbesar RAM Android
Aplikasi Android untuk menambah RAM
Judulnya terlalu mengejutkan ya ? sebenarnya tidak begitu juga karena RAM pada perangkat android terutama hp dan tablet android tidak bisa diperbesar begitu saja. Aplikasi ini sebenarnya memiliki fungsi hanya untuk mengatur penggunaan RAM, mana aplikasi yang harus berjalan dan mana aplkasi yang tidak boleh jalan. Jadi Intinya untuk memaksimalkan penggunaan RAM sehingga tidak begitu berat dan terasa lebih besar dari sebelumnya. gitu aja, ga juga karena masih banyak lagi fungsi lainnya terutama yang berkaitan dengan RAM Android.
Hanya untuk Android yang sudah di ROOT
Perlu diperhatikan aplikasi ini hanya dapat di instal dn dijal;ankan pada perangkat android yang sudah di root atau sudah mendapatkan akses super user. yang belum tau Rooting silakan baca dulu : apa itu rooting
Cara menggunakan RAM Manager Pro
salah satu cara untuk membuat / memperbesar RAM Android yaitu dengan membuat virtual RAM pada kartu memori. berikut urutan yang bisa dicoba - resiko ditanggung anda sendiri - :
Centang Set on boot dan lock launcher in memory
Pilih tab Ekstra dan pilih Swap File di bagian paling bawah.
Pilih lagi penyimpanan yang akan jadi virtual ram saya rekomendasikan di Memori Eksternal / Kartu memory lebih bagus ukuran kartu memorynya gede misalkan 8 GB keatas jadi klo 2 GB dibuat sebagai virtual RAM makan masih kosong.
Di kolom Swap File Location tulis saja "/storage/extsdcard" tanpa tanda petik.
Pilih selanjutnya berapa ukuran swap file yang akan di virtualkan pilih saja 2048MB [ 2GB ].
Lalu Pilih Confirm maka proses Swap File akan berjalan, biasanya memerlukan waktu 15menit-1jam.
Maka sobat usahakan baterai penuh atau masih banyak daya agar proses berjalan dengan sukses.
Terakhir Reboot, biasanya bila sukses ditandai dengan adanya file bernama Swap File di memori card kita jadi jangan di hapus yah.
Finish, sekarang Ram android sudah longgar dan bebas mau install aplikasi banyak.

Secara sederhana Timelapse adalah video statis yang merekan sebuah aktifitas bergerak dalam kurun waktu tertentu yang dipercepat. Mungkin diantar kita pernah melihat video matahari terbenam yang dipercepat sedemikian rupa, atau video pergerakan awan, atau video proses merekahnya sekuntum bunga mawar dan video lainnya. Proses pembuatan video tersebut sering disebut TimeLapse. [ untuk penjelasan lebih lengkap bisa dibaca disini -> apa itu video timelapse ]

Selanjutnya beragam pengaturan yang tersdia bisa kita coba sendiri dalam aplikasi tersebut.
